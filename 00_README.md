# Base44 Superagents — Messaging Framework

This repository contains the full positioning research, messaging framework, and copy strategy for **Base44 Superagents**.

All documents were built using the following skills:
- `product-marketing-context` — foundational positioning document (`.agents/product-marketing-context.md`)
- `competitor-alternatives` — structured competitive analysis
- `marketing-psychology` — behavioral science applied to messaging
- `copywriting` — copy frameworks and principles
- `launch-strategy` — GTM application

## The One Thing

> The barrier to AI agents isn't desire. It's setup.  
> Remove the setup. The desire converts directly to users.

## North Star Positioning

**For:** Professionals who want an AI agent doing real work for them  
**Is:** A managed AI agent  
**That:** Starts working in under a minute — because Base44 handles all infrastructure, security, and integrations  
**Unlike:** Self-hosted agents (OpenClaw) that require hours of DevOps and ongoing maintenance
